Preparando uma área de trabalho

Neste passo a passo vamos configurar uma área de trabalho e para isto vamos utilizar o formato A4 que foi desenhado anteriormente

  • Inicie o AutoCAD

  • Na caixa de diálogo Create New Drawing, clique no botão Open Drawing --> selecionar o Exercício 1 (formato A4) 

  •  salve o arquivo em sua pasta de trabalho com o nome Exercício 2

Configurar: Unidades, estilos de textos, estilos de pontos, tipos de linhas, camadas, atribuir larguras às linhas e ver as larguras atribuídas

  • as unidades para trabalhar em milímetros; abra a caixa de diálogo de FORMAT --> Units--> milímeters;

  •  o estilo de texto: abra a caixa de diálogo de FORMAT --> Text Style em Style Name --> New: na caixa de diálogo  criar um novo estilo: Desenho; na Font --> Font Name --> escolha uma fonte a utilizar no desenho --> Ok para confirmar a configuração. Não mude altura de texto <0.00> porque o AutoCAD já tem uma altura padronizada de 2.5 mm e posteriormente terá que utilizar várias alturas que podem ser dadas e/ou configuradas diretamente na caixa de texto.

  • O estilo de pontos: abra a caixa de diálogo de FORMAT --> Point Style e escolha uma das opções de estilos de pontos que aparecem, assim como a porcentagem do tamanho --> Ok para confirmar a configuração. 

  •  O tipos de linhas: abra a caixa de diálogo de FORMAT --> Linetype, dê um clique no botão Load localizado na parte superior da caixa de diálogo, e uma segunda caixa de diálogo será aberta com diferentes tipos de linhas, destas vamos selecionar as seguintes:

  • Center2 -->OK 

Volte a dar um Load e faça a escolha de:

  • Dashet2 ou Hidden2 --->OK

De novo dê um Load e faça a escolha de:

  • Dot2 -->OK

Então a caixa de diálogo Linetype Manager fica configurada com as seguintes camadas:

 

  • Bylayes; (padrão do AutoCAD

  • ByBlock; (padrão do AutoCAD

  •  Center2;

  • Continuous; (padrão do AutoCAD)

  • Dot2;

  • Hidden2

  • 0 (padrão do AutoCAD)

As camadas (Layer): Vamos preparar as camadas para ser utilizadas nos desenhos posteriores, as mesmas serão configuradas em função dos tipos de linhas a utilizar e dos objetivos das mesmas. As técnicas de camadas ajudam muito na organização e modificação de seus desenhos. O AutoCAD permite um número ilimitado de camadas;

Exemplo: vamos configurar uma camadas para contorno visível, como <contínua>, camada para os contornos invisíveis, como <tracejada>, camada para linhas de centros de circunferências, como <centro>, camada para dimensionar os objetos, nomeada <cotagem>, uma camada para as representações dos materiais nas vistas com cortes chamada <hachura>, uma camada para as linhas auxiliares que será chamada de <auxiliar>, uma camada para <texto> e uma última para <formato>.   

1.    Escolha FORMAT --> Layer a partir do menu suspenso, dê um clique no ícone Layers da barra de Properties, ou digite LA --> enter;

2.    Dê um clique no botão Show Details localizado na parte superior da caixa de diálogo, onde poderão ser atribuídas a cor, a espessura da linha e o tipo de linha, uma outra opção será mostrada a seguir;

3.    Dê um clique no botão direito superior da caixa de diálogo New. Uma nova camada designada como Layer1 aparece destacada na caixa da lista, você deve mudar esse nome para <contínua>, para dar uma cor a esta camada pode dar um clique acima da palavra White que aparece como padrão e uma caixa de diálogo das cores será mostrada, na qual você pode mudar para a cor que quiser, por exemplo a cor azul; o tipo de linha a utilizar será continuous; agora dê um clique na palavra Default da coluna Lineweight e mude a espessura da linha para <0.30>.

4.    Repita o clique no botão direito superior da caixa de diálogo New. Uma nova camada designada como Layer2 aparece destacada na caixa da lista, você deve mudar esse nome para <tracejada>, para dar uma cor a esta camada dê um clique acima da palavra White que aparece como padrão e mude para uma cor 250 (cinza); o tipo de linha a utilizar será Hidden2; agora dê um clique na palavra Default da coluna Lineweight e mude a espessura da linha para <0.15>.

5.    Repita o clique no botão direito superior da caixa de diálogo New. Uma nova camada designada como Layer3 aparece destacada na caixa da lista, você deve mudar esse nome para <centro>, para dar uma cor a esta camada dê um clique na seta de color e escolha uma nova cor ou pode manter qualquer uma das anteriores; o tipo de linha a utilizar será center2; agora dê um clique na palavra Default da coluna Lineweight ou na seta de Lineweight e mude a espessura da linha para <0.05>.

6.    Selecione uma nova camada e mude o nome para <hachura>, dê uma cor verde a esta camada; o tipo de linha a utilizar será continuous; e mude a espessura da linha para <0.05>.

7.    Selecione uma nova camada e mude o nome para <cotagem>, dê uma cor magenta a esta camada; o tipo de linha a utilizar será continuous; e mude a espessura da linha para <0.09>.

8.    Selecione uma nova camada e mude o nome para <auxiliar>, dê uma cor vermelha a esta camada; o tipo de linha a utilizar será dot2; e a espessura da linha pode ficar default.

9.    Selecione uma nova camada e mude o nome para <texto>, pode manter a cor white; o tipo de linha a utilizar será continuous; e a espessura da linha pode ficar default.

10.   Selecione uma nova camada e mude o nome para <formato>, pode manter a cor white; o tipo de linha a utilizar será continuous; e a espessura da linha pode ficar default.

Agora vamos atribuir larguras diferentes para o formato: Modify --> Properties, uma caixa diálogo será aberta:

1.    clique na divisória Categorized no alto da caixa de diálogo. Ela mostra as propriedades de acordo as categorias de propriedade.

2.    selecione o marco interno do formato, assim como as linhas verticais e a primeira linha horizontal (altura 50 mm);

3.    clique na opção Lineweight e dê uma espessura de 0.30;

4.    selecione o marco externo do formato, assim como as linhas horizontais que ficaram sem atribuir espessura e;

5.    clique na opção Lineweight e dê uma espessura de 0.09;

6.    selecione o texto do formato, e clique na opção Layer, note que aparece uma flecha à direita no nome da camada da opção Layer, clique na flecha e deve aparecer a lista das camadas configuradas, selecione a camada TEXTO. 

Para ver as larguras atribuídas no desenho do formato A4, dê um clique na opção LWT na barra de status (parte inferior da tela)  Por último deve salvar seu desenho com todas as configurações como FORMATO A4, para depois utilizar nos Trabalhos Prático planejados.
