Desenhando as vistas ortográficas

No seguinte passo a passo vamos utilizar a barra de ferramenta DRAW e a barra de ferramenta MODIFY,  entre outras, desenhando as três vistas ortográficas principais do seguinte modelo isométrico . 

  • inicie o AutoCAD

  • na caixa de diálogo Create New Drawing, clique no botão Open Drawing --> selecionar Formato A4.

  • salve o arquivo em sua pasta de trabalho com o nome Exercício 3

Segundo o capítulo de Desenho Projetivo vamos desenhar as três vistas principais desta perspectiva isométrica utilizando os comando de Draw -->Line e Draw -->Circle, e utilizar e configurar os comandos modificadores Modify -->Chamfer e Modify-->Fillet entre outros comando!

Agora vamos aprender usar os comandos modificadores Fillet e Chamfer; o primeiro permite fazer os raios de 5 mm e o segundo fazer os chanfros de 5 mm,situados  na base de altura 12 mm.

Estes comandos podem ser acionados diretamente do ícone da barra flutuante Modify ou do menu ModifyàFillet ou ModifyàChamfer

Começarmos com Fillet para fazer os raios de 5 mm

Command: _fillet

Current settings: Mode = TRIM, Radius = 10.0000

Primeiro, configurar o raio para 5mm a seguir: 

Select first object or [Polyline/Radius/Trim]: R à enter

Specify fillet radius <10.0000>: 5 -->enter para sair da configuração do raio e volte a dar enter para retomar o comando configurado (em caso de ser um raio de 10 mm, não precissa configurar e pode facer diretamente)

Current settings: Mode = TRIM, Radius = 5.0

Select first object or [Polyline/Radius/Trim]: dê um clique acima de uma das linhas

Select second object: dê um clique acima da outra linha perpendicular para formar o raio, e pronto! Já está o raio!

Agora você vai fazer sozinho o outro raio de 5 mm!!!

 Primeiro, configurar o Chanfro para 5mm a seguir

Command: CHAMFER (TRIM mode) Current chamfer Dist1=10.0000, Dist2=10.0

Select first line or [Polyline/Distance/Angle/Trim/Method]: d

Specify first chamfer distance <10.0>: 5

Specify second chamfer distance <5.0>: --> enter para sair da configuração do chanfro e volte a dar enter para retomar o comando configurado (em caso de ser um chanfro de 10 mm, não precisa configurar e pode fazer diretamente)

Command: CHAMFER (TRIM mode) Current chamfer Dist1=5.0, Dist2=5.0

Select first line or [Polyline/Distance/Angle/Trim/Method]: dê um clique acima de uma das linhas

Select second line: dê um clique acima da outra linha perpendicular à primeira para formar o chanfro.

 Faça sozinho o outro chanfro de 5 mm!!!

 Desenhe as outras duas vistas da figura seguindo o mesmo procedimento e segundo o desenho mostrado.

 Vamos atribuir espessuras diferentes para as figuras desenhadas: Modify --> Properties, uma caixa diálogo será aberta:

1.   clique na etiqueta Categorized no alto da caixa de diálogo. Ela mostra as propriedades de acordo as categorias de propriedade.

2.   selecione todas as linhas contínuas da figura, ou seja todas as linhas visíveis de seu desenho;

3.   clique na opção Layer e procure a camada contínua;

4.   selecione as linhas de centro de circunferência;

5.   clique na opção Layer e procure a camada centro. 

Para ver as espessuras atribuídas no desenho da figura, dê um clique na opção LWT na barra de status (parte inferior da tela)  

Por último deve salvar seu desenho com todas as configurações como Exercício 3.
