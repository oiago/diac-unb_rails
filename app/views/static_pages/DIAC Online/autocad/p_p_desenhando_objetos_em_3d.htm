Os comandos necessários para fazer um sólido em 3D, partindo de uma figura em 2D.

O comando que vamos utilizar para fazer um sólido é o comando Extrude, que pode ser acionado em Draw --> Solid --> Extrude, ou pode-se digitar ext e dar enter.

Seção 1: Command: EXTRUDE

  • Select objects: 1 found

  • Specify height of extrusion or [Path]: 5 enter 

  • Specify angle of taper for extrusion <0>:enter

Visualize seu desenho em isométrico View --> 3D Views--> SW  Isometric

 

Seção 2: Command: EXTRUDE

  • Select objects: 1 found

  • Specify height of extrusion or [Path]: 23  enter 

  • Specify angle of taper for extrusion <0>: enter

Visualize seu desenho em isométrico View --> 3D Views--> SW  Isometric

 

Seção 3: Command: EXTRUDE

  • Select objects: 1 found

  • Specify height of extrusion or [Path]: 13  enter 

  • Specify angle of taper for extrusion <0>: enter

Visualize seu desenho em isométrico View --> 3D Views--> SW  Isometric

 

Seção 4: Command: EXTRUDE

  • Select objects: 1 found

  • Specify height of extrusion or [Path]: 8  enter 

  • Specify angle of taper for extrusion <0>: enter

Visualize seu desenho em isométrico View --> 3D Views--> SW  Isometric

 

Por último deve unir todas as seções com um comando modificador, para isso, selecione no menu principal  Modify-->Solids Editing-->Union, selecione cada seção e dê OK.
