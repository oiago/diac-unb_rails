Edição e Modificação de Desenho

     O primeiro conceito que deve ficar bem claro é que para alterar qualquer desenho, temos que especificar a ação e o objeto, ou objetos que sofrerão a ação. Exemplo: copiar um círculo, copiar é a ação e o círculo é o objeto que será copiado. Portanto, toda ação implica numa seleção de objeto. Esta seleção pode ser feita de diversas formas: selecionando primeiro o objeto e depois dar o comando de ação ou dar o comando de ação e depois selecionar o objeto. Sempre que você executar um comando de edição, receberá a seguinte mensagem na linha de comando: Select objects: I found

Isto quer dizer que o AutoCAD encontrou um objeto, o selecionou e pediu para selecionar outros objetos, se não quer selecionar mais nenhum objeto tecle Enter para encerrar a seleção e executar o comando dado. (Também são válidos como Enter a barra de espaço e o botão direito do mouse).   

A barra de ferramenta MODIFY é acionada na barra de menu principal, ou na barra de menu flutuante MODIFY, ou no menu lateral.  

Dê um clique em cada nome de comando da barra de ferramenta flutuante Modify

 

Erase:

Elimina os objetos criados. Os mesmos podem ser eliminados de forma individual, fazendo a seleção e depois dar enter com o botão direito do mouse ou com o teclado, ou também pode ser feita uma janela nos objetos que se quer selecionar e dar enter. Em caso de ter selecionado um objeto por engano, pode segurar Shift e dar um clique no objeto. O AutoCAD permite a eliminação de objetos com a tecla DEL. Os objetos apagados podem ser recuperados pelo comando UNDO ou Oops, sendo que o último recupera apenas a última seleção removida.    

inicio

Copy:

Executa a duplicação simples ou em seqüência de um ou mais objetos de desenho. Primeiro seleciona-se o objeto a copiar, em seguida define-se o ponto de referência (ponto base) e por último, o deslocamento, que pode ser informado por coordenadas pelo teclado, aleatoriamente na tela ou com o uso da ferramenta de precisão Osnap. Em caso de necessitar múltiplas copias, deve digitar Màenter e depois definir o ponto de referência (ponto base) e por último, o deslocamento.  

inicio

Mirror:

Este comando produz um espelhamento dos objetos selecionados.

  • Yes (Y): define que os objetos selecionados para o espelhamento serão apagados.

  • No(N): define que os objetos selecionados para o espelhamento não serão apagados.

Utilize a variável (digitar) MIRRTEXT para controlar o espelhamento de detalhes que contenham texto. Se atribuirmos o valor <1> (default), os textos são invertidos após o espelhamento. Digitando um valor 0 (zero) não inverte os textos. Em caso de haver textos, esta opção deve ser configurada antes de acionar o comando.  

inicio

Offset:

Permite a duplicação com distância digitada via teclado, ou utilizando ferramentas Osnap. Este comando pode ser usado em circunferências, linhas arcos, polilinhas, elipses e outros comandos de desenho. Quando usado em segmentos de retas, duplica os mesmos paralelamente. Quando em figuras curvas duplica-os concentricamente.  

inicio

Array:

Este comando permite duplicar objetos regularmente em formato retangular ou circular, conforme a opção escolhida.

  • Retangular: (digitar R) Distribui um elemento em M linhas e N colunas, a uma distância X entre as colunas e a uma distância Y entre as linhas. Após a seleção de objetos, a opção para a escolha da forma de duplicação é habilitada.

  • Polar: (digitar P) copia um ou mais objetos N vezes em torno de um ponto central, preenchendo um ângulo a ser informado pelo usuário. Se o ângulo de preenchimento for positivo, os objetos serão copiados no sentido anti-horário, caso contrario são copiados no sentido horário.

Existe a opção circular, acionada pela letra C, um pouco diferente da polar, copia objetos circularmente, solicitando o número de itens e o ângulo entre eles, depois de informado o ângulo de rotação dos objetos, pode-se definir se os objetos sofrerão ou não, rotação em torno de seu próprio eixo.      

inicio

Move:

 Este comando possibilita o deslocamento de um ou mais objetos dentro de um desenho. Esse deslocamento pode ser controlado pelo movimento do mouse, por coordenadas ou pelo uso de ferramentas de precisão Osnap. Sempre que usarmos o Osnap será necessário informar o ponto de base de forma totalmente precisa. No caso de usar coordenadas, pode-se escolher qualquer ponto da tela, como base (Base Point) do deslocamento.  

inicio

Rotate:

Este comando executa uma simples rotação de objeto selecionado em torno de um ponto base. Depois de selecionado o ponto base, a opção para especificar de uma referencia torna-se disponível. Com a opção Reference, podemos informar o ângulo final desejado para uma determinada peça, ao invés do ângulo a rotacionar.   

inicio

Scale:

Aumenta ou diminui os objetos selecionados de igual forma nas direções dos eixos X, Y e Z.

Quando utilizamos um fator de escala, mudamos todas as dimensões do objeto selecionado. Um fator de escala maior que 1, aumenta o objeto. Um fator de escala menor que 1, encurta o objeto.  

inicio

Stretch:

Este comando deforma as entidades pelos pontos finais ou vértices selecionados. Comando utilizado nos casos de alterações parciais do dimensionamento de desenhos. Dependendo da forma de utilização, tem o mesmo efeito do comando Move.  

inicio

Lengthen:

Este comando possibilita uma variação no comprimento de objetos abertos como linhas, polilinhas, elipse e arcos.

  • Delta: permite especificar um incremento linear ou angular para retas e arcos.  

  • Percent: Altera o comprimento de retas e arcos a partir de um percentual de seus comprimentos.

  • Total: Altera o comprimento de uma entidade, sendo solicitado o dimensionamento final.

  • Dynamic: Permite uma alteração visual do comprimento controlada pelo mouse.  

inicio

Trim:

Este comando remove partes de objetos (linhas, circunferências, arcos, polylines, etc) quebrados no ponto de intersecção, com outros objetos previamente selecionados. Selecionamos inicialmente os objetos cortantes e a serem cortados e confirmamos a seleção com enter, em seguida, selecionamos os segmentos dos objetos que deveram ser cortados.  

inicio

Extend:

Este comando prolonga um objeto até um limite definido por outro objeto. Primeiro passo é selecionar o objeto limite, confirma-se a seleção com enter (também pode ser com o botão direito do mouse ou com a barra de espaço) e posteriormente dê um clique no objeto a ser extendido.

inicio

 

Break:

O comando Break, quebra a entidade selecionada, gerando duas outras entidades ou simplesmente apagando parte da mesma. Quando o comando Break é usado para a quebra de somente um ponto, não funciona para entidades fechadas como são: circunferências, elipses e spline. Objetos como circunferências, elipses e splines são sempre quebrados em sentido anti-horário.

Após selecionado o objeto a quebrar, pode-se acionar a opção First Point, que permite informar o primeiro e o segundo ponto de quebra. Digitando o símbolo arroba @ como segundo ponto, o AutoCAD entende que a quebra deve ocorrer somente no primeiro ponto informado. 

inicio

Chamfer:

Este comando cria chanfros através da informação de duas distâncias e um ângulo. As distâncias são referentes ao prolongamento das linhas que se encontram num determinado vértice. Para criar vértices entre duas linhas (cantos vivos), basta executar o comando normalmente utilizando ambas as distâncias igual a 0 (zero). Opções para criar chanfros:  

  • Polyline: quando aplicado a uma polyline, constrói chanfros em todos os cantos possíveis.

  • Distance: permite definir uma distância de chanfros.  

  • Angle: permite definir um ângulo de chanfro, se este for o caso.  

  • Trim: configura o AutoCAD para cortar ou não, as linhas a serem chanfradas.  

  • Method: permite definir o método a ser utilizado para o chanfro (distância e ângulo ou duas distâncias).  

inicio

Fillet:

Este comando produz arcos de circunferência entre duas entidades. Também pode gerar cantos vivos se o raio definido for igual a 0. Pode ser utilizado em linhas paralelas. Na opção Polyline (digitar P) quando aplicado sobre uma polilinha, arredonda todos os cantos de uma só vez. Na opção Radius (digitar R), é para definir o valor de raio a ser utilizado. Na opção Trim (digitar T), define se os objetos selecionados serão ou não cortados após o comando Fillet.    

inicio   

Explode:

Este comando desagrupa entidades compostas como polilinhas, blocos, cotas hachuras, etc. Dependendo da estrutura do objeto em edição, serão necessários vários comandos Explode para que se tenha uma determinada entidade totalmente independente. Exemplo é quando explodido um bloco que tem cotas, primeiro tem que explodir o bloco e depois a cota.
