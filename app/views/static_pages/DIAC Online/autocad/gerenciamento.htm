Configurações no AutoCAD

Neste capítulo vamos configurar nossa área de desenho: os limites do desenho, os tamanhos dos formatos padronizados pelas normas ABNT, o texto, as unidades, as camadas, veremos também, no caso de utilizar uma escala determinada para plotar (imprimir), conheceremos como configurar o texto para na saída ter o tamanho previsto, utilização dos recursos de controle de imagem, Zoom, e mostraremos qual é a barra de status e como funciona, e tem mais...       

Os comandos de gerenciamento podem ser agrupados em:

  • Comandos Auxiliares: Estes comandos definem as unidades, os limites do desenho e estabelecem uma orientação para a execução dos elementos dos desenhos. Os comandos unidades, e limites do desenho, estão no menu suspenso FORMat  Units, Thickness e Drawing Limits (Unidades, Espessuras e Limites do Desenho).

Os modos do AutoCAD são utilizados como ferramentas de desenho, estes modos se encontram na barra de status mostrada na tela de entrada do AutoCAD. Se quiser saber mais clique aqui para saber como configurar os mesmos. 

  • Comandos de precisão: Estes comandos determinam a seleção de pontos de construção do desenho com precisão, ativando uma atração automática a um ponto do objeto (ponto notável). Esta capacidade é conhecida como SNAP (saltos) ou OSNAP (Object Snap). Eles podem ser acionados pelo menu lateral (****), pelo menu suspenso Viewà Toolbarsà Object Snap ou Toolsà Drafting Settingsà Object Snap, ou ainda pelo menu cursor acionando Shift em conjunto com o botão direito do mouse. Este comando pode ser ativado ou desativado com F3.

  • Comandos de armazenamento; Estes comandos permitem salvar os desenhos executados no AutoCAD, enquanto eles se encontram na memória RAM do computador. Se este for desligado antes de salvar, todo o trabalho é perdido.       

     As opções são igual que no Windows:

       - Save: salva o desenho atual para um arquivo ativo (aberto);  

       - Save as: salva o desenho atual para um arquivo com nome diferente;    

       - Exit: encerra o desenho atual e sai para o sistema operacional, caso haja alterações no desenho não salvas, antes de sair aparece a caixa de  diálogo.  

inicio

  • comandos Layer ou camadas; O AutoCAD possibilita a execução de um desenho em diferentes níveis, facilitando assim o trabalho do usuário, uma vez que desenhos complexos podem ser separados em níveis, que se apresentam como se fossem papeis transparentes perfeitamente superpostos. Desta maneira cada parte do desenho pode ser desenhado num nível. A utilização de níveis possibilita que um mesmo desenho possua várias versões, uma em cada nível. Apenas o desenho ativo pode ser plotado. É aconselhável a determinação de uma cor diferente para cada nível, para facilitar a visualização de cada nível na tela. Ao iniciarmos um desenho o AutoCAD nos leva para o nível 0, sendo que este nível não pode ser apagado nem renomeado. A criação e manutenção dos níveis são efetuados através da caixa de diálogo Layer Properties Manager" que pode ser ativada via menu de barra --> "Format"--> Layer; menu de tela --> FORMAT -->Layer ou no ícone da barra "Object Properties"

  • Controle de Imagem: Os comandos de controle de imagem operam somente a nível de visualização do desenho, não alterando as características das elementos criadas. Estes comandos estão no menu suspenso  View > ZOOM, Pan e Aerial View;  

  • Outros Comandos: 

       Comando Redraw   

Este comando redesenha a imagem presente na tela, isto é,  faz uma limpeza na imagem da tela, eliminando as marcas auxiliares,(blips). Ao emitir este comando a tela dará uma "piscada" e o desenho é atualizado. View --> Redraw

      Comando Regen

Este comando provoca uma regeneração forçada do desenho inteiro, redesenhando os elementos na tela refinando a representação dos objetos, no entanto recomenda-se a utilização deste comando quando a representação estiver muito grosseira. View--> Regen--> Regen . Existe também o comando Regen all que atua em todas as janelas (viewports) abertas na tela do monitor.

Configurar o tamanho do texto na saída da plotagem

Para estes exemplos, vamos considerar que você quer que na saída de impressão que o TEXTO  no desenho fique numa altura de 3 mm, para isto, você vai multiplicar o tamanho do texto escolhido pelo fator de escala utilizado na plotagem. Ou seja: quer um texto impresso de 3 mm e está utilizando um fator de escala de 5 (1:5) então deve colocar um tamanho de texto de 15 (multiplica vezes o fator de escala da plotagem 5).

Barra de Status

    Snap : permite um deslocamento de forma incremental do cursor de desenho na tela. Este comando opera em conjunto com o Grip podendo ou não ter o mesmo valor. Estes deslocamentos podem ser configurados para outras distâncias escolhidas pelo desenhista em Tool--> Drafting Setting -->Snap and Grid Pode ativar ou desativar com F9. 

   Grid: estabelece uma grade auxiliar na tela do desenho criando um padrão de pontos uniformemente espaçados. Estes espaçamentos podem ser configurados para outras distâncias escolhidas pelo desenhista em Tool--> Drafting Setting --> Snap and Grid. Pode ativar ou desativar com F7.  

  Ortho : este modo força as linhas de desenho a ser estritamente vertical e horizontal, também restringe o desenho de arcos. Quando ativado o modo Ortho o modo Polar Tracking é automaticamente desligado. Pode ativar ou desativar o modo Ortho dê um clique em F8.

    Polar Tracking: Configura as coordenadas polares fixando um incremento do ângulo e os ângulos adicionais, e quando ligada, vai mostrando na tela a distância da linha e o ângulo que a mesma está sendo movida.  Pode ser ativada ou desativada com F10.  

 

    Agora veremos como configurar seu desenho para trabalhar com as ferramentas Polar Tracking e Polar Snap. A opção Polar Tracking auxilia você a alinhar seu cursor segundo ângulos horizontal e vertical exatos, como com a régua T e esquadros. Polar Snap é semelhante a GRID SNAP quanto a fazer com que o cursor se mova em incrementos exatos. A principal diferença é que  Polar Snap só funciona em conjunto com Polar Tracking.

  Como configurar Polar Snap

  • Abra a caixa de diálogo Drafting Settings novamente (Tool--> Drafting Setting);

  • Certifique-se de ter selecionado a divisória Snap and Grid. Dê um clique no botão Polar Snap no grupo de botões Snap Type & Style;

  • Dê um clique duplo na configuração Polar Distance e digite 10 (se as unidades estão configuradas em mm, em caso de estar configuradas em centímetros digite 1) como definição de Polar Snap;

  • Dê um clique em Ok para sair da caixa de diálogo  Drafting Settings.

    Ao mover o cursor sobre a área de desenho, observe que o modo Snap aparece desativado. Quando a opção Polar Snap está ativada você só terá o efeito snap efetivo ao desenhar um objeto.  Você pode, não entanto, comutar entre Grid Snap e Polar Snap rapidamente.

       Comutar entre Grid Snap e Polar Snap

  • Dê um clique na ferramenta Line do menu Draw, ou digite L àenter;

  • Dê um clique com o botão direito do mouse no botão SNAP na barra de status;

  • Selecione Grid Snap On no menu suspenso. Isto coloca a você no modo Grid Snap.

  • Pressione F6 até que veja as coordenadas x,y serem atualizadas dinamicamente quando mover o cursor. Você pode clicar diretamente no leitor de coordenadas para percorrer as opções de leitura entre estática, dinâmica e polar. Observe que ao mover o cursor. Observe que, ao mover o cursor, o leitor de coordenadas exibe o cursor variando por incrementos cujo valor definiu na configuração de Grid Snap.

  Comandos de precisão 

  • Endpoint:[endp] seleciona todos os pontos finais de retas, polilinhas, arcos, curvas e vértices de faces 3D.

  • Midpoint:[mid] seleciona o ponto médio de uma reta ou de um arco. No caso de polilinha seleciona o ponto médio de um segmento de polilinha.

  • Center:[cen] seleciona o centro de um arco ou um círculo. Você deve clicar sobre o arco ou círculo e não sobre o seu centro aparente.

  • Node:[nod] seleciona um objeto pontual.

  • Quadpoint:[qua] seleciona o ponto cardeal mais próximo (norte, sul, leste ou oeste)

  • Intersection:[int] seleciona a interseção de objetos.

  • Extension:[ext] seleciona um ponto que esteja alinhado com uma extensão imaginária de uma reta. Por exemplo, você pode escolher um ponto no espaço que esteja realmente sobre essa reta. Para usar esse ponto digite extà enter durante a seleção do ponto ou escolha Extension a partir do menu instantâneo, em seguida mova o cursor para a linha cuja extensão você deseja usar e mantenha o cursor aí até que você veja um pequeno marcador em forma de cruz sobre a linha.    

  • Insert:[ins] seleciona o ponto de inserção de texto, bloco, Xrefs e camadas.

  • Perpendicular:[per] seleciona uma posição sobre um objeto que é perpendicular ao ùltimo ponto escolhido.

  • Tangent:[tan] seleciona um ponto sobre um arco ou um círculo que representa a tangente a partir do último ponto escolhido.

  • Nearest:[nea] seleciona um ponto sobre um objeto mais próximo do ponto escolhido.

  • Apparent intersection:[apint] seleciona a interseção aparente de dois objetos. Isto é ùtil quando você deseja selecionar a interseção de dois objetos que na realidade não se interceptam. Para isto será pedido que selecione os dois objetos.  

  • Parallel:[par] permite selecionar um objeto que seja paralelo a outro segmento de reta existente. Para usar esta opção digite --> enter, em seguida mova o cursor sobre a linha em relação a qual você deseja obter uma paralela e mantenha o cursor até que veja um pequeno marcador em forma de cruz sobre a reta.

Comando Layer:

  • Na caixa de diálogo Layer Properties Manager" encontramos "named layer filter" que é um filtro dos níveis do desenho; o botão "new" podemos criar um novo nível ou camada; o botão "current" possibilita tornar um determinado nível como sendo o ativo; os botões "Show e Hide Details" mostra ou esconde os detalhes característicos do nível selecionado.

  • Lock, Color, Linetype, Lineweight, Plot Style, Plot, Active Viewport Freeze e New Viewport Freeze.

  • On: quando a lámpada está na cor amarela significa que o respectivo nível está ativado e é visível na tela gráfica., no caso que estiver desativada está de cor cinza e esse nível não aparece  visível na tela.

  • Freeze in All Viewports: congela/descongela a camada em todas as Viewports;

  • Active Viewport Freeze: congela/descongela a camada apenas na Viewport ativa;

  • New Viewport Freeze: congela/descongela  o nível ou camada em todas as novas Viewports;

  • Lock (cadeado fechado): é destinado para a ação de ativar ou desativar edições num nível ou camada o nível está impossibilitado de receber edições, enquanto Unlock (cadeado aberto) o nível pode receber edições.

  • Color: selecionamos uma cor  que será associada a um nível.  

  • Linetype: podemos carregar e definir o tipo de linha utilizado nesse nível ou camada.

  • Lineweight: podemos escolher a espessura da linha a ser utilizado nesse nível ou camada.

  • Plot: nos permite plotar ou não os elementos de um determinado nível.

Controle de Imagem:

  • Comando Zoom

Este comando atua como se o observador estivesse afastando-se ou aproximando-se do desenho, e ele pode ser acionado via teclado, via ícone ou pelo menu suspenso VIEW--> Zoom --> Realtime ou Previus ou Windows ou

 ZOOM 

[Realtime]: muda a visualização automaticamente através da movimentação do mouse,

    - para abaixo--> afastamento

    - para cima--> aproximação

[Previus]: restaura a janela anterior.

[Windows]: define nova janela através de dois pontos na diagonal

[Dynamic]: define nova janela dinamicamente. O comando apresenta uma tela mostrando:

    - um retângulo azul pontilhado representando o limite do desenho;

    - um retângulo verde pontilhado representando a janela corrente;

    - um retângulo móvel  representando a próxima janela.  

    [Scale x/xp]: modifica a janela mudando a escala da visualização corrente, gerando efeito de aproximação e afastamento. Xp só é utilizado no modo "paper space"  Exemplo:

- "2x" : aumenta duas vezes o tamanho atual do desenho;

- "0.5x" : diminui duas vezes o tamanho atual do desenho;

- "2" : aumenta duas vezes o tamanho original do desenho;

- "0.5": diminui duas vezes o tamanho original do desenho.

[Center] : altera o posicionamento do centro da janela corrente ou seja a distância do observador ao ponto marcado, a menor distância vamos aproximar o ponto marcado e a maior distância nos afastamos do ponto.

[In]: aproxima os objetos : Zoom 2x

[Out]: afasta os objetos : Zoom 0.5x

[All]: encaixa os limites do desenho na tela gráfica, quando o desenho ultrapassa os limites da tela gráfica ele será exibido por inteiro e se estivermos com o Grip ativado, podemos visualizar onde estão os limites ativos do desenho.

[Extents] : Ajusta a janela de forma a enquadrar todos os objetos.

[Pan]: permite modificar a região visível do desenho apenas deslocando a janela corrente, mantendo suas proporções a escala.

 [Aerial View] : é uma ferramenta de navegação  que mostra o desenho por inteiro em uma janela separada e permite especificar rapidamente qual parte deve ser localizada e exibida.
