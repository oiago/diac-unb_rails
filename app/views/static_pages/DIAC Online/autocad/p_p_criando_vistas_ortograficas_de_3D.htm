Criando a projeção ortogonal das vistas ortográficas a partir do modelo em 3D 

Quando é desenhado o modelo sólido em 3D, podemos ver as vistas ortográficas através da opção Viewports no menu  View, aqui podem ser mostrados as três vistas ortográficas além do modelo isométrico, nestes desenhos aparecem todas as linhas contínua, mas quando é dado um Hide, podemos ver que as linhas que ficam dentro do modelo e que estão além da vista observada desaparecem e não é possível mostrá-las como realmente devem ser desenhadas, tracejadas, para conseguir isto vamos aprender agora a criar estas vistas ortográficas a partir do modelo sólido.

Em caso de não ter carregado a linha Hidden como linha tracejada, deve carrega-la para poder ver as mudanças que serão feitas no modelo, e para isto dê um clique em Format -->Linetype--> no botão load dê um clique e escolha a linha Hidden -->Ok para que fique configurada.

 1.      Estamos com um desenho no model e passamos o mesmo para o paper fazendo um clique no layout1 ou layout2 vejam a seguinte figura no modo MODEL:

 2.      Agora passamos para o layout1 e estamos mostrando o mesmo modelo no (espaço do papel) paper space, mas, neste caso foram abertas 4 janelas de visualização para mostrar as vistas ortográficas e o isométrico do modelo. Pode-se ver que as vistas aparecem sem interligação de tamanho porque cada janela aparece numa escala ajustada, no capítulo de plotagem pode ver como colocar todas as vistas na escala de impressão, bem alinhada ou seja dependentes e interligadas como devem ficar num projeto.

 

3.      Vamos passar do modo paper para Floating Model Space dando um clique duplo na janela de visualização da vista frontal;

4.      Escolha Setup Profile na palheta de ferramenta Solid, ou escolha Draw-->Solid-->Setup-->Profile.

5.      Dê um clique na vista frontal do modelo sólido e em seguida pressione  enter para confirmar

6.      No prompt: Display hidden profile lines on separate layer? [Yes/No] <Y>: (Exibir a silhueta das linhas ocultas em camada separadas?) pressione enter para confirmar

7.      No prompt: Project profile lines onto a plane? [Yes/No] <Y>: (projetar a silhueta das linhas sobre um plano?) pressione enter para confirmar

8.      No prompt: Delete tangential edges? [Yes/No] <Y>: (Apagar as arestas tangenciais) pressione enter para confirmar

9.      Para mostrar as mudanças da ferramenta Setup Profile, torne invisível a camada do modelo sólido.

10.  Abra a ferramenta Layer Properties Manager, ou escolha Format --> Layer na barra de menu. Selecione a camada cujo nome comece com o prefixo PH.

11.  Mude o tipo de linha para Hidden (em caso de não aparecer com este tipo de linha), e observe que a vista frontal do modelo exibe agora as linhas ocultas como linhas tracejadas.

12.  Faça o mesmo procedimento para cada vista ortográfica e desenhe as linhas de centros de circunferências onde for necessário para que suas vistas fiquem totalmente representadas como mostra a seguinte figura.

 

A ferramenta  Setup Profile cria um desenho 2D a partir do seu modelo em 3D. Este desenho em 2D é projetado sobre um plano imaginário que é paralelo à vista a partir da qual você selecionou o modelo ao usar a ferramenta Setup Profile, para ver isto claramente dê um clique no seu modelo no modo Model Space.

1.      Dê um clique na divisória Model para ir para o modo Model Space.

2.      Ative a camada do modelo sólido e verá a vista em 2D projetada próximo ao modelo em 3D.
