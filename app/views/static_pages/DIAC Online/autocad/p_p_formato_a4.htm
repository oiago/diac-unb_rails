Desenhando um formato A4 por coordenadas

Em este exercício será desenhado um formato A4 utilizando o comando “Line” no menu Draw.

As dimensões gerais do formato são: marco externo 210 x 297; e marco interno 180x287.

 1.      Marco externo: em coordenadas cartesianas: Comand: line ou digite L

a)     Pto 1: 0,0;

b)     Pto 2: 210,0;

c)      Pto 3: 210,297;

d)     Pto 4: 0,297;

e)     Pto 5: 0,0;   

2.      Marco Interno em coordenadas relativas: Comand: line ou digite L

a)     Pto 1: 25, 5;

b)     Pto 2: @180, 0;  ou @180<0;

c)      Pto 3: @ 287<90;

d)     Pto 4: @180<180;

e)     Pto 5: @287< -90;  

Este formato pode ser desenhado também com o comando rectangle utilizando o Pto 1= (0,0,0) e o Pto 2 = (210,297) (sem parênteses) ; e para fazer o retângulo interno utilize o Pto 1 =(25,5) e o Pto 2: (205,292) para fechar o retângulo. Tente!  

3.      Agora vamos fazer a legenda deste formato utilizando o comando Line e combinando os tipos de coordenadas.

Linhas horizontais:

a)     Pto 1: 25, 25;  àenter para confirmar posição

b)     Pto 2: @180, 0;  ou @180<0; à enter para confirmar posição e enter para sair do comando;  

para retomar o comando pode dar enter novamente

c)      Pto 3: 25, 40; àenter para confirmar posição

d)     Pto 4: @180, 0;  ou @180<0; à enter para confirmar posição e enter para sair do comando  

para retomar o comando pode dar enter novamente

e)     Pto 5: 25, 55; àenter para confirmar posição

f)        Pto 6: @180, 0;  ou @180<0; à enter para confirmar posição e enter para sair do comando;

 

Linhas verticais:

g)     Pto 1:70,5; àenter para confirmar posição

h)      Pto 2: @50 <90; à enter para confirmar posição e enter para sair do comando;  

para retomar o comando pode dar enter novamente

i)        Pto 3: 160,5; àenter para confirmar posição

j)        Pto 4: @50<90; àenter para confirmar posição e  enter para sair do comando;
